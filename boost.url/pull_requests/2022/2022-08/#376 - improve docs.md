> Username: vinniefalco  
> Created_at: 2022-08-09 15:27:12 UTC  
> Updated_at: 2022-08-09 15:27:30 UTC  
> Url: https://github.com/boostorg/url/pull/376#discussion_r941485144  

Consider reordering this from shortest -> longest. Having "url" sandwiched in the middle doesn't look very good. We should also add authority view:  
```  
* __url__: A modifiable URL.  
* __url_view__: A non-modifiable reference to a URL.  
* __static_url__: A modifiable URL with fixed capacity, inline storage.  
* __authority_view__: Technically not a URL but still often useful.  
```  
  
And then we might say:  
> These containers maintain a useful invariant: they always contain a valid URL (or authority, for the last one)
