> Username: sdarwin  
> Created at: 2023-10-15 22:04:47 UTC

> Url: https://github.com/boostorg/serialization/issues/300  

We are working on updating the boost.org website, and when displaying the documentation of this boost library in that environment the display breaks due to the fact that `<frame>`s are used in the html.  There will probably be a work-around, such as opening the docs in a new browser tab, but it would be preferable to either convert these docs to AsciiDoc as a modernization, or otherwise remove `<frame>`.
