Hey, new user of spirit here. First, I must say that this is one of the most impressive libraries I've seen in my humble 5 years of programming.  
I've followed the tutorials, but I'm stuck on trying to implement [X3 Program Structure](https://www.boost.org/doc/libs/develop/libs/spirit/doc/x3/html/spirit_x3/tutorials/minimal.html) on [Annotations - Decorating the ASTs](https://www.boost.org/doc/libs/develop/libs/spirit/doc/x3/html/spirit_x3/tutorials/annotation.html). [Last section (Config)](https://www.boost.org/doc/libs/develop/libs/spirit/doc/x3/html/spirit_x3/tutorials/annotation.html#spirit_x3.tutorials.annotation.config) briefly mentions how this should be done, but the jump from the context definition:  
```c++  
using context_type = x3::phrase_parse_context<x3::ascii::space_type>::type;  
```  
to   
```c++  
using phrase_context_type = x3::phrase_parse_context<x3::ascii::space_type>::type;  
  
typedef x3::context<error_handler_tag,  
       std::reference_wrapper<position_cache>,  
       phrase_context_type>  
context_type;  
```  
is not explained. `error_handler_tag` is not explained and simply declaring a class results in an error where `x3::get<position_cache_tag>(context).get()` does not compile since the function expects 1 argument. A novice user like myself might get confused here and get stuck unable to resolve the error.
