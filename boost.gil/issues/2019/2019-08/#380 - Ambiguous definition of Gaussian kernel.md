There seems to be diverging definitions of Gaussian kernel.  
  
The one that GIL follows is from [Wikipedia article](https://en.wikipedia.org/wiki/Gaussian_blur).  
  
The diverging contestant is [a calculator site](http://dev.theomader.com/gaussian-kernel-calculator/) ranked pretty high in google search for "Gaussian kernel calculator".  
  
The values generated by the two diverge, and pretty significantly, but the application in at least one other algorithm have shown that what GIL uses is correct. What the site generates though seem to be *normalized* values. It is important to get some more sources to put clear distinction and clearly define the two, so we could use this thread as reference in future discussions.

I found the definition of normalized Gaussian: it is a kernel where all entries sum to 1. On this page: https://en.wikipedia.org/wiki/Gaussian_function there are plots of various normalized versions, and taking integral will amount to one in each case. Further more, from [the site](http://dev.theomader.com/gaussian-kernel-calculator/), I tweaked both sigma and size, and each time sum was very close to one. Link to spreadsheet: [link](https://docs.google.com/spreadsheets/d/1wgF2_GiDIQZOOA27mshUjNWh-VBqvsVwFYA_5fOlwcs/edit?usp=sharing)
