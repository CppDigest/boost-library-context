## Comment 6

> Created at: 2019-07-14 15:52:51 UTC

The number of mutexes in the mutex table is bounded: https://github.com/boostorg/asio/blob/develop/include/boost/asio/detail/strand_executor_service.hpp#L120 so not every `strand` object will cause "allocated memory growth".
