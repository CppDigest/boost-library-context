With the goal of making Hana more lightweight, it should be possible to include only the algorithms one wants. To achieve this, the algorithms will be split into single headers. Right now, all the algorithms working on Sequences are in the `<boost/hana/sequence.hpp>` header. The goal is to have a separate header for each algorithm, and then perhaps have `<boost/hana/sequence.hpp>` include them all (but not necessarily). Some concepts have already been modularized.
