This PR adjusts some `@boost` links in old documentation files that were also counting on the git layout instead of the boost release layout, like the links we fixed in https://github.com/boostorg/utility/pull/86.   
  
We can check the links are now correct with `b2 libs/utility/doc` from the boost root directory.   
  
This PR also makes the boost.utility documentation match the latest boost.utility patch in https://github.com/boostorg/website/tree/master/doc/libs.
