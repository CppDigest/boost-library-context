Hi, you have to include `#include <boost/redis/src.hpp>` in one of your `.cpp` files, or create a new file e.g. `boost_redis.cpp` that includes it. Then compile this file normally with the rest of your code. The `src.hpp` contains all `*.ipp` files. For an example see  
  
https://github.com/boostorg/redis/blob/46525371b9247a1ec208a6157880ec77f5ddadc4/examples/boost_redis.cpp
