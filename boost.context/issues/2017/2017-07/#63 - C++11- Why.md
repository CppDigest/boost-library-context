call/cc, the public API of boost.context, requires C++11 feauters. call/cc is used as foundation (context switching) in boost.coroutine2 and boost.fiber.  
  
ASIO can be used boost.coroutine (pre C++11) and boost.coroutine2 (C++11 ...). For boost.coroutine2 integration a pull request is pending in boost.asio (https://github.com/boostorg/asio/pull/52).  
  
boost.coroutine is deprected because:  
- pre C++11  
- convoluted implementation  
- uses internal API from boost.context
