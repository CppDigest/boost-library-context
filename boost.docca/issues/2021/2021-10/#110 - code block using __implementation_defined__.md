The `__implementation_defined__` identifier in code blocks should be formatted with italics as _implementation-defined_:  
  
![image](https://user-images.githubusercontent.com/1503976/139596064-bc7e8dd4-add1-4521-bafe-73abe6120e99.png)  
  
To escape to Quickbook in a code block using double backticks, e.g.:  
```  
constexpr ``['implementation-defined]`` alnum_chars;  
```
