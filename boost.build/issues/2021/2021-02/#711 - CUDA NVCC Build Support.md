Several issues have been found in the boost modules when building boost headers in CUDA's NVCC compiler. NVCC is a ubiquitous C++ compiler in the field GPGPU computing and boosts applications overlap with CUDA considerably. Support for NVCC in boost build and regression testing would ensure these issues don't continue on the NVCC platform.
