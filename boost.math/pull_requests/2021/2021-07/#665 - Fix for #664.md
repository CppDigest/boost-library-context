I think that's 0, both empirically and intuitively, right? (but also actually independent of the variance, if I understand what you're asking)  
  
```Mathematica  
x = Table[  
   {  
    N@var,  
    Correlation[  
     RandomVariate[NormalDistribution[0, var], 100000],  
     RandomVariate[NormalDistribution[0, var], 100000]  
     ]  
    },  
   {var, 10^Range[0, -10, -1]}  
   ];  
x // TableForm  
```
