> Url: https://github.com/boostorg/math/issues/571#issuecomment-812524687  

> but the compiler should short-circuit evaluation  
  
If you talking about preprocessor that not was like that, it was fixed in the standard not a long time ago too, because of that we had to do the check this way:  
```  
#ifdef __has_include  
# if __has_include(somethig)  
...  
```  
  
See https://gcc.gnu.org/onlinedocs/cpp/_005f_005fhas_005finclude.html and the paper that introduced `__has_include` itself https://wg21.link/p0061
