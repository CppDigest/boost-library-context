Continuing on my investigations, I've modified `select_freelist::type` to use boost aligned allocator instead of the default one.  
  
```c++  
    typedef typename mpl::if_c<IsCompileTimeSized || IsFixedSize,  
                               fixed_size_freelist<T, fixed_sized_storage_type>,  
                               freelist_stack<T, boost::alignment::aligned_allocator_adaptor<Alloc, BOOST_LOCKFREE_CACHELINE_BYTES > >  
                              >::type type;  
```  
  
It seems to fix random deadlocks and crashes I had in [HPX ](https://github.com/STEllAR-GROUP/hpx) 1d_stencil_8 example.
